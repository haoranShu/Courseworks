	// For example:

	Key "f": full screen mode
	Key "c": exit full screen mode
	Key "a": rotate about X-axis for 30 degrees
	Key "b": rotate about Y-axis for 30 degrees
	Key "c": rotate about Z-axis for 30 degrees
	Key "x": translate to X+ a unit
	Key "y": translate to Y+ a unit
	Key "z": translate to Z+ a unit
	Key "X"("shift + x"): translate to X- a unit
	Key "Y"("shift + y"): translate to Y- a unit
	Key "Z"("shift + z"): translate to Z- a unit
	Key "l": scale larger to 1.1x
	Key "s": scale smaller to 0.91x
	Key "Left": increase camera position.x
	Key "Right": decrease camera position.x
	Key "Up": increase camera position.y
	Key "Down": decrease camera position.y
	
	mouse left click and drag: move the center of perspective projection matrix
	
	mouse wheel: adjust the field of view
